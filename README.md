# Projeto Integrador: Ciência de dados aplicada a situações de mercado

Este PI tem como objetivo explorar as aplicações da ciência de dados para situações de mercado, utilizando os conhecimentos aprendidos no curso até agora, com ênfase na relação com bancos de dados e gerenciamento de projetos.

Após clonar o repositório, o venv (ambiente virtual Python) pode ser inicializado com auxílio do pacote [uv](https://github.com/astral-sh/uv). Após sua instalação (através das orientações oferecidas no README do pacote - recomendamos utilizar `pip`), simplesmente execute `uv sync`. O ambiente estará pronto para desenvolvimento e execução.
